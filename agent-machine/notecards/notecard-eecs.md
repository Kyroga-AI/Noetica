# eecs — canonical reference (authored canon, the exam note card)


## ac
- **Impedance**: `Z = R + j (omega L - 1/(omega C))`

## circuits
- **Ohm's law**: `V = I R`
- **Kirchhoff's voltage law**: `sum of voltages around a loop = 0`
- **Kirchhoff's current law**: `sum of currents into a node = 0`
- **Electrical power**: `P = V I = I^2 R = V^2 / R`
- **Capacitor i-v**: `i = C dv/dt`
- **Inductor i-v**: `v = L di/dt`
- **Thevenin equivalent**: `any linear network = V_th in series with R_th`

## digital
- **De Morgan's laws**: `not(A and B) = notA or notB`

## dsp
- **Nyquist rate**: `fs >= 2 fmax`

## signals
- **Transfer function**: `H(s) = Y(s) / X(s)`
- **Fourier transform**: `X(f) = integral x(t) exp(-j 2 pi f t) dt`
- **Laplace transform**: `X(s) = integral x(t) exp(-s t) dt`

## transient
- **RC time constant**: `tau = R C`
