# chemistry — canonical reference (authored canon, the exam note card)


## acids
- **pH**: `pH = -log[H+]`
- **Water autoionization**: `Kw = [H+][OH-] = 1.0e-14`
- **Henderson-Hasselbalch**: `pH = pKa + log([A-]/[HA])`

## electrochem
- **Nernst equation**: `E = E0 - (R T / n F) ln(Q)`

## equilibrium
- **Equilibrium constant**: `K = [products] / [reactants]`

## gases
- **Ideal gas law**: `P V = n R T`
- **Boyle's law**: `P1 V1 = P2 V2`
- **Charles's law**: `V1 / T1 = V2 / T2`

## general
- **Density**: `d = m / V`

## kinetics
- **Rate law**: `rate = k [A]^m [B]^n`
- **Arrhenius**: `k = A exp(-Ea / (R T))`
- **First-order integrated rate**: `ln[A] = ln[A]0 - k t`
- **First-order half-life**: `t_half = ln(2) / k`

## quantum
- **Quantization (Bohr)**: `E_n = -13.6 eV / n^2`
- **Photon energy**: `E = h nu = h c / lambda`

## solutions
- **Molarity**: `M = mol solute / L solution`
- **Dilution**: `M1 V1 = M2 V2`
- **Raoult's law**: `P = x_solvent P0`
- **Freezing-point depression**: `dTf = i Kf m`

## spectroscopy
- **Beer-Lambert law**: `A = epsilon l c`

## thermo
- **Gibbs free energy**: `dG = dH - T dS`
- **Gibbs and K**: `dG = -R T ln(K)`
- **Heat**: `q = m c dT`
- **Hess's law**: `dH_rxn = sum dH_products - sum dH_reactants`
- **Entropy change**: `dS = q_rev / T`
- **First law**: `dU = q + w`
