# physics — canonical reference (authored canon, the exam note card)


## analytical_mechanics
- **Lagrangian**: `L = T - V`
- **Euler-Lagrange equation**: `d/dt(dL/dqdot) - dL/dq = 0`
- **Hamiltonian**: `H = sum(p qdot) - L = T + V`
- **Hamilton's equations**: `qdot = dH/dp, pdot = -dH/dq`
- **Action**: `S = integral L dt`

## circuits
- **Ohm's law**: `V = I R`
- **Electrical power**: `P = I V = I^2 R`
- **RC time constant**: `tau = R C`

## circular
- **Centripetal acceleration**: `ac = v^2 / r`

## electromagnetism
- **Gauss's law (E)**: `div E = rho / epsilon0`
- **Gauss's law (B)**: `div B = 0`
- **Faraday's law**: `curl E = -dB/dt`
- **Ampere-Maxwell law**: `curl B = mu0 J + mu0 epsilon0 dE/dt`
- **Lorentz force**: `F = q (E + v x B)`

## electrostatics
- **Coulomb's law**: `F = k q1 q2 / r^2`

## energy
- **Work**: `W = F d cos(theta)`
- **Kinetic energy**: `KE = (1/2) m v^2`
- **Gravitational PE**: `PE = m g h`
- **Spring PE**: `PE = (1/2) k x^2`
- **Power**: `P = W / t = F v`

## gravitation
- **Newton's gravitation**: `Fg = G m1 m2 / r^2`

## kinematics
- **Velocity (constant a)**: `v = v0 + a t`
- **Position (constant a)**: `x = x0 + v0 t + (1/2) a t^2`
- **Velocity-displacement**: `v^2 = v0^2 + 2 a (x - x0)`

## mechanics
- **Newton's second law**: `F = m a`
- **Weight**: `Fg = m g`
- **Friction**: `Ff = mu N`

## momentum
- **Momentum**: `p = m v`
- **Impulse**: `J = F t = dp`

## oscillation
- **Hooke's law**: `F = -k x`

## quantum
- **Time-dependent Schrodinger equation**: `i hbar dpsi/dt = H psi`
- **Time-independent Schrodinger equation**: `H psi = E psi`
- **Quantum Hamiltonian**: `H = -hbar^2/(2m) nabla^2 + V`
- **Wavefunction normalization**: `integral |psi|^2 dx = 1`
- **Expectation value**: `<A> = integral psi* A psi dx`
- **Heisenberg uncertainty**: `dx dp >= hbar/2`
- **de Broglie wavelength**: `lambda = h / p`
- **Planck relation**: `E = hbar omega = h f`
- **Canonical commutator**: `[x, p] = i hbar`
- **Infinite square well energies**: `E_n = n^2 pi^2 hbar^2 / (2 m L^2)`
- **Quantum harmonic oscillator**: `E_n = hbar omega (n + 1/2)`

## relativity
- **Lorentz factor**: `gamma = 1 / sqrt(1 - v^2/c^2)`
- **Energy-momentum relation**: `E^2 = (p c)^2 + (m c^2)^2`
- **Time dilation**: `dt = gamma dt0`

## statistical
- **Partition function**: `Z = sum exp(-E_i / (k T))`
- **Boltzmann distribution**: `P_i = exp(-E_i / (k T)) / Z`
- **Boltzmann entropy**: `S = k ln(Omega)`
- **Helmholtz free energy**: `F = -k T ln(Z)`
- **Fermi-Dirac distribution**: `n = 1 / (exp((E - mu)/(k T)) + 1)`
- **Bose-Einstein distribution**: `n = 1 / (exp((E - mu)/(k T)) - 1)`
