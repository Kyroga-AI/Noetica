# biology — canonical reference (authored canon, the exam note card)


## biochemistry
- **Michaelis-Menten**: `v = Vmax [S] / (Km + [S])`

## cells
- **Surface area to volume**: `ratio = SA / V`

## ecology
- **Exponential growth**: `dN/dt = r N`
- **Logistic growth**: `dN/dt = r N (K - N) / K`

## energetics
- **Gibbs free energy**: `dG = dH - T dS`
- **ATP / free energy**: `dG = dG0 + RT ln(Q)`

## genetics
- **Hardy-Weinberg allele**: `p + q = 1`
- **Hardy-Weinberg genotype**: `p^2 + 2 p q + q^2 = 1`
- **Monohybrid ratio**: `Aa x Aa -> 3:1 phenotype, 1:2:1 genotype`

## molecular
- **Central dogma**: `DNA -> (transcription) RNA -> (translation) protein`

## physiology
- **Water potential**: `Psi = Psi_p + Psi_s`
- **Solute potential**: `Psi_s = -i C R T`
- **Nernst (membrane potential)**: `E = (RT/zF) ln([ion]_out / [ion]_in)`

## stats
- **Chi-square**: `chi^2 = sum( (O - E)^2 / E )`
